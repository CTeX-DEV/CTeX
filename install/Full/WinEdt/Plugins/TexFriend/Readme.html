使用

一般操作

只需点击鼠标,选中符号所对应用的LaTeX命令就会插入到WinEdt编辑的当前文本的光标所在处.此外,系统还会检查该LaTeX文件是否调用了相应的宏包.若没有,则会在 \documentclass...下面加入一行,如 \usepackage{...}.

为使该系统正确工作,LaTeX文件必须以\documentclass开始.对于CCT中文TeX,建议使用
\documentclass[11pt,CJK]{cctart}
\documentclass[11pt,CJK]{cctbook}
\documentclass[11pt,CJK]{cctpage}
... ...

选则字体

有两种方式
1. 直接点击相应字体命令,WinEdt会在当前位置插入\textrm{}或\songti之类的命令
2. 先选中一段文字,再点字体命令, 这时该字体命令仅对选中的文字有效,即选中的部分变成了\textrm{selected words}或 {\songti 选中的文字}

文件格式说明

您可以按以下方法扩充字符集,从而加强TeXFriend的功能.

[fonts.txt]

每行定义了一个字符集。其中caption为字符集的名称，bitmap为存放这些字符的位图文件（可以为其他格式），该图被分成r x c个小图，每个小图代表一个字符，h,w分别为小图的高和宽。command为一个文本文件，它的每一行定义了一个字符串，与bitmap中的小图相对应，它的格式为 分类号：命令 其中分类号必须为1位字符，冒号之后为任意字符串

[TexFriend.ini]

class与command定义了对上述字符串的解释。例如， class="1" command="Ins" 表示对"1"类命令， 使用WinEdt宏Ins. 例如， 当点击鼠标时，若相应位置的字符串为"1:\bbalpha", 则系统会调用WinEdt执行命令 Ins('\alpha') 即把\alpha插入当前位置。

[fonts.txt]

edt为可选参数，若存在，应该对应于一个WinEdt宏命令文件，该文件应存放在%WinEdtPath%\bin\MikTex文件夹。 例如，对于AMS符号， edt="amssymb.edt", 点击鼠标时，除插入相应字符串以外，还执行文件amssymb.edt. 该文件用于检查LaTeX文件中是否调用了宏包"amssymb.sty", 若没有,则会在\documentclass...下面加入一行\usepackage{amssymb}.
